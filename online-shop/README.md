My JavaScript Frameworks Assignment (Online Shop)

A responsive online shop built with **Next.js, React, and TypeScript**.  
The application fetches products from an external API and allows users to browse products, view details, search items, add products to a cart, and complete a checkout flow.

This project was built as part of a **JavaScript Frameworks course assignment**.

---

Features

- Product list fetched from an external API
- Product detail pages
- Dynamic product search
- Shopping cart system
- Add / remove items from cart
- Adjust product quantities
- Cart total calculation
- Checkout success page
- Contact form with validation
- Toast notifications for cart actions
- Responsive design for desktop, tablet, and mobile

---

Tech Stack

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**
- **React Context API** (cart state)
- **React Hot Toast** (notifications)

---

API

Product data is fetched from the Noroff Online Shop API.

Endpoint used:
- GET https://v2.api.noroff.dev/online-shop
- GET https://v2.api.noroff.dev/online-shop/{id}


API documentation:  
https://docs.noroff.dev/docs/v2/basic/online-shop
